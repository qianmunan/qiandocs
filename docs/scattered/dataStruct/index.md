## 我的想法
数据结构在编写大型软件或项目中用处非常大。而个人写的小型项目，可能感受不到。而作为一个想要成为一个码农的人来说，不学习一下数据结构与算法，总感觉少了一点什么。而且在日常编写代码的过程中，总感觉少了点什么，所以在这里开了一个文档，来记录一下我学习数据结构与算法的过程。希望自己不负此行，在结束最后一个文档的编写后，能有所收获，并运用到自己的日常中来。

## 我的学习想法
+ 通过阅读数据结构与算法相关数据，将一些概念的东西记录下来
+ 将书上例子先理解下来，自己在自己的计算机上实现一下（注意编码规范）
+ 利用好Leetcode这个工具，多做一些算法相关的题目。
+ 注意算法的优化
